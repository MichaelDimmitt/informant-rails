# Overview

## Informant Rails

The informant-rails gem provides Rails and ActiveRecord hooks for The Informant.

[![Homepage](http://www.informantapp.com)](http://www.informantapp.com)

## Compatibility

Fabrication is tested against Ruby 1.9.3, 2.0.0, and Rubinius.
